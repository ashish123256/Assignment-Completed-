# Tech Stack

**Client:** React, Redux, 

**Server:** Node, Express, Axios, Mongoose

**Database:** MongoDB




**For Client**
```bash
  cd client
```
Start the server

```bash
  npm start
```
**For Server**
```bash
  cd server
```
Start the server

```bash
  npm run dev
```


# Install Dependencies

**For Backend** -`cd server` `npm i`

**For Frontend** - `cd client` ` npm i`
